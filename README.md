🛢️ Oil Well Data Analysis & Visualization
📌 Project Overview

This project focuses on exploratory data analysis (EDA) and visualization of oil well data. The goal is to extract meaningful insights from production and operational data using statistical analysis and graphical visualization techniques.

The project does not include machine learning modeling, but instead emphasizes:

Understanding data patterns 📊
Identifying trends in oil production ⛽
Detecting anomalies and operational behavior ⚠️
Supporting decision-making through visualization
🧠 Objective

Oil well performance data is complex and influenced by many factors. This project aims to:

Explore and visualize oil well data to better understand production behavior, reservoir characteristics, and operational efficiency.

📊 Dataset Description

The dataset contains oil well information such as:

🔹 Well Information
Well ID
Location / Field
Well type (oil, gas, injection)
🔹 Reservoir & Geological Data
Reservoir pressure
Depth
Porosity
Permeability
Formation type
🔹 Production Data
Oil production rate (bbl/day)
Gas production rate
Water cut (%)
Cumulative production
🔹 Operational Parameters
Tubing pressure
Bottom hole pressure
Choke size
Flow rate
🔍 Data Analysis Performed
📈 1. Production Analysis
Oil production trends across wells
High vs low producing wells
Production distribution
📊 2. Statistical Analysis
Mean, median, variance of production rates
Correlation between variables
Outlier detection
🧪 3. Reservoir Insights
Relationship between reservoir properties and production
Effect of water cut on oil output
⚠️ 4. Data Quality Check
Missing values handling
Outlier identification
Data consistency checks
📉 Visualizations Used
Line plots (production trends over time)
Bar charts (well comparison)
Scatter plots (relationship analysis)
Heatmaps (correlation matrix)
Histograms (distribution of variables)
Box plots (outlier detection)
💡 Key Insights
Production varies significantly across different wells
Higher water cut generally reduces oil output
Reservoir pressure strongly influences production rates
Some wells show consistent decline trends over time
A small number of wells contribute majority of production
🛠️ Tools & Libraries
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
📌 Conclusion

This project demonstrates how exploratory data analysis and visualization can be used to understand oil well performance. The insights gained can support better operational decisions and highlight important production patterns without the use of machine learning models.
